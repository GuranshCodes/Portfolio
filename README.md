<h2>Features</h2>

<ul>
  <li>Responsive design for desktop and mobile devices</li>
  <li>Interactive project showcase</li>
  <li>Smooth animations using Framer Motion</li>
  <li>Skills and technologies section</li>
  <li>Contact section</li>
  <li>Fast performance with Vite</li>
  <li>Modern UI built with Tailwind CSS</li>
</ul>

<h2>Installation</h2>

<h3>Clone the repository</h3>

<pre><code>git clone https://github.com/GuranshCodes/Portfolio.git
cd Portfolio
</code></pre>

<h3>Install dependencies</h3>

<pre><code>npm install
</code></pre>

<h3>Run the development server</h3>

<pre><code>npm run dev
</code></pre>

<h3>Create a production build</h3>

<pre><code>npm run build
</code></pre>
